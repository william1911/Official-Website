    User Information

    How to become a staff member?

    • Must be active on the server you are applying for.
    • It helps to be active on Teamspeak / Discord before you are given staff.
    • Apply here: Application Link
    • Being mature and know staff common sense.

    Chain of Command

    Network-Owner > Network-Manager > Manager > Admin/Supervisor > Moderator > Chat-Mod

    Duties of each rank.

    Network-Owner:

    • Makes sure the state of the network is good.
    • Makes sure all server bills are payed.
    • Gives tasks to Network-Managers.
    • Makes sure the Managers of each server are doing there jobs

    Network-Manager:

    • Usually helps the Network-Owner(s) pick and choose Managers of a server.
    • Helps a Manager choose appropriate staff for the server.
    • Report to the Network-Owner(s) anytime there is staff abuse/Manager not doing their job.

    Server-Manager:

    • Makes sure the state of the server they run is good.
    • Makes server rules.
    • Makes sure all Plugins are in working condition.
    • Has the ability to ask Network-Owner(s) for server advertising.
    • Approves staff applications.
    • Interacts with players on the server and takes suggestions to improve the server.

    Admin/Supervisor:

    • Follows what tasks the Manager gives them.
    • Has the ability to report lower staff and the manager of abuse/insubordination.
    • Interacts with the players of the server.
    • Makes sure that lower staff are doing their job.

    Moderator:

    • Keeps the players in check.
    • Enforces server rules.
    • Keeps chat clean and friendly.
    • Reports to higher staff of anything suspicious going on in the server.

    Chat-Mod:

    • Watches chat.
    • Keeps chat clean and friendly.

    Requirements to be staff:

    Manager:

    • MUST communicate via Teamspeak / Discord & Kik.
    • MUST be active on the server and Teamspeak / Discord.
    • MUST have a good reputation throughout the server/network.

    Admin/Supervisor:

    • MUST communicate via Teamspeak / Discord.
    • Be active on Teamspeak / Discord
    • Should have a Kik. (If not its okay)
    • Been a chat-mod and a Mod.
    • Be active on the server. (Usually 3 days a week unless told otherwise)

    Moderator:

    • MUST communicate via Teamspeak / Discord
    • Has the ability to come on Teamspeak / Discord if needed.
    • Been a Chat-Mod.
    • Be active on the server. (Usually 3 days a week unless told otherwise)

    Chat-Mod:

    • MUST have Teamspeak / Discord.
    • Has the ability to come on Teamspeak / Discord if needed.
    • Be active on the server. (Usually 3 days a week unless told otherwise)

    How to get promoted:

    Member --> Chat-Mod:

    • Apply
    • Be active on the server
    • DO NOT ask about your application.
    • Use Teamspeak / Discord.

    Chat-Mod --> Mod:

    • Do a good job as a Chat-Mod.
    • Be active on the server.
    • Use Teamspeak / Discord.
    • Have minimum abuse complaints.

    Mod --> Admin:

    • Do a good job as a Mod.
    • Be active on the server.
    • Use Teamspeak / Discord.
    • Have minimum abuse complaints.

    Important Notes:

    • Using Teamspeak / Discord is very important as it is the main form of communication we use here at Digital Hazards.
    • Be friendly to players, player's trust is a very important factor when we promote.
    • Be active, being active shows that you are interested in the server.
    • Be patient, we do not go through our applications everyday, only when we need staff (Do NOT ask)
